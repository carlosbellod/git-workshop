## Hotfix branch from a tag

 1. Create the branch **hotfixes/hotfix1** from tag **Release1.0**
 2. Fix the issue
 3. Commit & push
 4. Swicth to master and merge **hotfixes/hotfix1**
 5. Delete local & remote **hotfixes/hotfix1** branch
